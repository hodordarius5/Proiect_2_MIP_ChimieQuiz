# ChimieQuiz

ChimieQuiz is a .NET MAUI educational application designed to test chemistry knowledge using multiple-choice quizzes.

## Features
- Chapter-based quizzes
- 5-option multiple-choice questions
- Final score and percentage
- Review of incorrect answers
- Retry wrong questions
- Clean and modern UI

## Technologies
- .NET MAUI
- C#
- XAML
- JSON (local data storage)

## Project context
This project was developed as part of the MIP course and demonstrates:
- Cross-platform app development
- UI design with XAML
- Navigation and data handling
- Use of AI-assisted development
